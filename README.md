# Cronómetro de Competidores

## Descripción

Esta aplicación web permite registrar el tiempo de llegada de varios competidores utilizando un cronómetro.  
El sistema calcula automáticamente el tiempo de cada competidor y muestra la diferencia respecto al primer lugar.

La aplicación fue desarrollada utilizando **HTML, CSS y JavaScript** y está publicada mediante **GitHub Pages**.

## Cómo funciona la aplicación

El sistema simula una competencia donde se registran los tiempos de llegada de los participantes.

### Pasos de uso

1. Presionar el botón **Iniciar** para comenzar el cronómetro.
2. Escribir el nombre del competidor en el campo de texto.
3. Presionar **Registrar tiempo** cuando el competidor llegue.
4. El sistema mostrará en la tabla:
   - Nombre del competidor
   - Tiempo registrado
   - Diferencia con el primer lugar
5. El competidor con el menor tiempo será marcado como **Ganador**.
6. Para iniciar una nueva carrera se puede presionar **Reiniciar**.

## Funcionalidades
